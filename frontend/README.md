# Inventory Management System

**Inventory Management System** is a full-stack application built with **React (Vite)** for the frontend, **Node.js + Express** for the backend, and **SQLite** for the database. The system allows users to **add, update, delete, and view inventory items**, with additional features like **search functionality** and **low-stock warnings**. The frontend is responsive and interactive, while the backend handles API requests and database operations efficiently. This project was built using my React knowledge and backend implementation guided by **online tutorials and YouTube**.

## Features
- Add, update, delete, and view inventory items
- Search functionality to quickly find products
- Low-stock warning to alert when item quantity is low
- Responsive and interactive frontend
- Backend API handling and database operations with SQLite

## Tech Stack
- Frontend: React (Vite), HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: SQLite
